Auth

This is a demo of a backend for authentication, built using Express.

The server generates a JSON Web Token (JWT) using the jsonwebtoken libary. When a user creates an account or logs in, the server signs a JWT and sends it back to the client. The client stores the JWT, and sends it in the Authorization request header of subsequent requests to the server.

Passwords are secured using bcrypt password-hashing.

This is a demo, and the JWT secret key is not secure. Do not use real credentials that you use on other accounts.

For the purposes of this demo, the JWT expiration time is set to 30 seconds. When the JWT has expired, you will be logged out.
